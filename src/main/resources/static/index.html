Приложение для знакомств

Synxo помогает знакомиться с людьми, у которых реально есть что-то общее с тобой.

Сначала ты создаешь профиль и выбираешь интересы из понятного списка. Потом видишь в ленте только тех, с кем уже есть пересечения. Общение открывается только после взаимной симпатии, так что переписки не начинаются случайно и без ответа.

Лента

Не случайные профили, а люди с похожими интересами

Лента собирается на основе выбранных интересов и показывает общие точки соприкосновения прямо в карточке.

Взаимность

Чат открывается только тогда, когда симпатия обоюдная

Сначала лайк, потом взаимность, и только после этого появляется полноценный диалог.

Профиль

Редактирование анкеты, фото и состояния в одном месте

Профиль можно менять в любой момент: обновлять интересы, загружать фото и выходить из аккаунта оттуда же.
